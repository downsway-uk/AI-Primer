# 🤖 AI-Primer

Welcome to the **AI-Primer** repository! 

This repository is a curated collection of documents, templates, and training resources designed to demystify artificial intelligence. It explains the mechanics behind Large Language Models (LLMs) like ChatGPT and Google Gemini, while providing practical pathways for both personal learning and enterprise development.

## 🎯 What's in this Repository?

Whether you are an AI enthusiast, a developer, or an enterprise professional, this repository serves as a starting point for understanding and deploying AI. The materials focus on:

* **Understanding LLMs:** How popular models operate under the hood.
* **Locally Hosted AI:** Guides for running models on your own desktop or laptop.
* **Open Source Tools:** Links and resources for free-for-personal-use AI tools.
* **Cloud Hosted Resources:** Platforms and tools for learning and building.
* **Enterprise Development:** Frameworks and resources for building real-world AI applications.
